Objectives

So far I have a simple 2 column, 3 row grid. I'd like to use it as a boilerplate for starting any 3 row, 2 column page I'd like to build in this "website". I know the design is real sparse here and that's important because this website is really just about figuring some things out once and for all, and even getting more and more complicated as I go. When I've master this, I'll start adding different layouts, more complicated components, like columns (in the footer perhaps?), cards, modals and navbars etc. Then I'll perhaps go even further and add some different page types like image gallery pages, hero pages, etc.
