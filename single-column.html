Objectives

So far I have a simple 2 column, 3 row grid. I'd like to use it as a boilerplate for starting any 3 row, 2 column page I'd like to build in this "website". When I've master this, I'll start adding different layouts, more complicated components, like columns (in the footer perhaps?), cards, modals and navbars etc. Then I'll perhaps go even further and add some different page types like image gallery pages, hero pages, etc.
